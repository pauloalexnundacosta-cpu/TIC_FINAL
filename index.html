Sou finalista do curso Técnico de Gestão e Programação de Sistemas Informáticos, onde ao longo de 3 anos adquiri competências sólidas em programação, bases de dados, redes e desenvolvimento web.

FAQ

Perguntas Frequentes

Respostas às dúvidas mais comuns sobre o meu trabalho e processo.

Que tecnologias utilizas nos teus projetos?
Trabalho principalmente com HTML5, CSS3 e JavaScript para o Front-End. Também tenho conhecimentos em PHP, SQL (bases de dados), Python e C#. Para design, utilizo o Figma e princípios de design responsivo.
Como organizas o teu trabalho individual?
Sigo um processo estruturado: planeamento (definição de requisitos e esboço), estruturação semântica do HTML, estilização com CSS organizado por componentes e revisão final com testes em vários dispositivos. Mantenho o código limpo, comentado e versionado com Git.
Fazes sites apenas com HTML e CSS, sem frameworks?
Sim! Este site que estás a ver foi construído exclusivamente com HTML e CSS puros, sem qualquer framework ou JavaScript para a navegação. Acredito que dominar as bases é essencial antes de utilizar frameworks. Consigo entregar sites completos e funcionais apenas com HTML e CSS.
Os teus sites são responsivos?
Sim, sempre. Todos os sites que crio são testados em telemóveis, tablets e desktops. Utilizo media queries, unidades relativas e layouts flexíveis (Flexbox/Grid) para garantir uma experiência consistente em qualquer ecrã.
Quanto tempo leva a criar um site?
Depende da complexidade do projeto. Um site institucional simples (como este) pode levar 1 a 3 dias. Projetos mais complexos com funcionalidades dinâmicas podem levar algumas semanas. O importante é garantir qualidade em cada etapa.
Que tipo de projetos já fizeste?
Durante o curso, desenvolvi landing pages, dashboards, blogs, protótipos de e-commerce, formulários multi-step, galerias e sites institucionais. Todos aplicando boas práticas de HTML semântico, CSS organizado e design responsivo.
Estás disponível para trabalho freelance?
Sim! Estou disponível para projetos freelance, estágios e colaborações. Se tens um projeto em mente, entra em contacto através da página de Contactos e podemos discutir como posso ajudar.
